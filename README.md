## Objective

- Understand when and why components re-render.
- Analyze the impact of memoization (`React.memo`, `useCallback`, `useMemo`).
- Visualize the component tree, render frequency, and potential optimizations.
- Provide an embeddable panel (`<RenderInsightPanel />`) and a demo playground.
